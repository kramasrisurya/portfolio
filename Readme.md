Rama Sri Surya Portfolio

This repository contains the source code for my professional portfolio website, showcasing my journey as a Computer Science student and aspiring Machine Learning Engineer.

Features:
Responsive Design: Optimized for various screen sizes including mobile and desktop.

Glassmorphism UI: Modern aesthetic using background blurs and backdrop filters.

Smooth Animations: Integrated fade-in effects for a dynamic user experience.

Detailed Sections: Includes Home, About, Projects, Resume, and Contact pages.

Built With:
HTML5: Semantic structure for web pages.

CSS3: Custom styling with Flexbox, Grid, and Keyframe animations.
